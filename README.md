### Compile and install

Для Windows:

	> cd radix
	> cmake -DCMAKE_BUILD_TYPE=RELEASE -G "Visual Studio 15 2017 Win64"

	Open radix\Radix.sln
	Build
	


### Плюсы проекта

		Простой и понятный интерфейс
		Не требует специфических знаний
		Полностью автоматизированное получение root-прав
		Универсальность
